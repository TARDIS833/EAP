# Logic 8_0

## 핵심 변경

1. 온라인 원버튼 패치 업데이트 추가
- `patch_source.json`의 `manifest_url`을 읽어 패치 매니페스트를 다운로드
- 파일별 SHA256 검증 후 `configs/`에 반영
- 반영 전 `backups/<timestamp>/configs` 자동 백업
- 반영 후 `load_mappings()` 재로딩

2. Cafe24 실무 양식 보정 강화
- `옵션` -> `주문상품명(옵션포함)` 자동 생성
- `핸드폰/전화번호`를 우선순위 기반 단일 `수령인 휴대전화`로 통합
- 중복 컬럼명 자동 제거

3. 7_9 이슈 반영
- `출고명`에 `; 사이즈=`가 섞이던 문제 수정
  - `색상/사이즈` 정규식 경계에 `;` 포함
  - 선택 토큰 제거 정규식 강화
- `핸드폰`, `수령인 휴대전화`가 Series 문자열로 깨지던 문제 수정
  - 중복 전화 컬럼 생성 방지

## 온라인 패치 운영

- 프로그램 폴더의 `patch_source.json`에 `manifest_url` 설정
- GUI의 `온라인 패치 업데이트` 버튼 클릭
- 성공 시 최신 규칙 반영 완료 메시지 출력
